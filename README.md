## Installing / Getting started

Clone repository:

```shell
$ git clone https://github.com/gabrieljony/challenge-cubos.git
$ cd challenge-cubos/
```

Install dependencies:

```shell
$ yarn install
```

## Development environment - http://localhost:3000/

```shell
$ yarn start
```
